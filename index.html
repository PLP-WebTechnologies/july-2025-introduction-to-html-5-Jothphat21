Images

Any images added should consider the following:

  1. Providing meaningful description by adding "alt" attribute to the image tag.
  2. Compressing images to an optimum size using width and height attributes.

Using Proper Headings

Headings should follow a logical order from <h1> to <h6> depending on the content's structure.

  • <h1> Used for the main title of the page
  • <h2> Used for subheadings under h1
  • <h3> Used for sub-subheadings, and so on
